Product Information sheet for: Guidare Boat Navigation Software Suite (Version: 1), model: Mercury.

Description.

The product Guidare Boat Navigation Software Suite is a software suite that includes a boat navigation system, a map database, and a web-based user interface. The boat navigation system provides users with real-time navigation information for their boats, including current speed, heading, and location. The map database includes detailed maps of various waterways around the world, allowing users to easily find their way around. The web-based user interface allows users to access their navigation data and maps from any computer with internet access.

Highlights.

Connect To Your Favorite Third Party Devices

This feature allows users to connect to their favorite devices, such as a GPS device, in order to view detailed navigation information. This is helpful for ensuring that the user receives the most accurate guidance possible while underway.

Navigate Any Waters With Preloaded Mapping And Coastal Charts

This feature allows users to navigate through waterways and coastal waters without having to manually enter data. The software suite includes preloaded mapping and charts that allow users to easily navigate their boat.

Manage Your Marine Experience From Nearly Anywhere

The Manage Your Marine Experience feature allows you to manage your marine experience from nearly anyplace in the Guidare Boat Navigation Software Suite. This includes accessing your boat’s current location, speed, heading, and other navigation data; configuring your boat’s settings; and managing your marine insurance policies.

Connect Traditional And Scanning Sonars

This feature allows users to connect their traditional sonars (e.g., an echo sounder) and scanning sonars (e.g., a multibeam echosounder) to the Guidare Boat Navigation software suite. This permits users to view data from both types of sonars simultaneously on the same screen, which can help them make more informed decisions when navigating.

Panoptix Sonar Support

The feature " in the Guidare Boat Navigation Software Suite" allows users to use a Panoptix sonar unit in their boat to improve navigation and assessment of their naval assets.

Bluechart And Vision Chart Support

The Bluechart and Vision Chart Support in the Guidare Boat Navigation Software Suite provides users with the ability to view charts and navigation data on a computer or mobile device. This feature allows users to plan their voyage using detailed maps and information, as well as track their progress using real-time visualizations.

Built In Wifi

The built in wifi in the Guidare Boat Navigation Software Suite allows users to connect to the internet and access the Guidare Boat Navigation software without connecting to a wireless network. This is useful for when there is no available wireless network or when the user wants to keep their wireless connection private.

Fishfinder

The in the Guidare Boat Navigation Software Suite is a software that allows the user to find and track fish in open water.
